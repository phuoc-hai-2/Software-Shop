# Frontend - Software Shop

ReactJS + Ant Design + Redux

## Modules
- Auth (Đăng ký/Đăng nhập)
- Trang sản phẩm, chi tiết sản phẩm
- Giỏ hàng, thanh toán
- Quản lý đơn hàng
- Đánh giá, hỗ trợ
- Trang quản trị (Admin)

## Getting Started
1. Cài Node.js
2. `npm install`
3. Cấu hình API endpoint trong `.env`
4. `npm start`
