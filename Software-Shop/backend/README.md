# Backend - Software Shop

Node.js (Express) + MongoDB (Mongoose)

## Modules
- Auth (JWT, Email)
- User Profile
- Product & Category
- Cart & Order
- Payment (VNPay)
- Review & Rating
- Chat/Support
- Admin Dashboard
- Notification & Email

## Getting Started
1. Cài Node.js, MongoDB
2. `npm install`
3. Cấu hình biến môi trường trong `.env`
4. `npm run dev`
